# Contributing

Thank you for your interest in this project.

## Before you contribute

This is an **unofficial design concept** for W Steak Restaurant. Please read [DISCLAIMER.md](DISCLAIMER.md) before submitting changes.

## What we welcome

- UI/UX improvements to the editorial concept design
- Accessibility fixes
- Performance optimisations
- Bug fixes in HTML/CSS/JS
- Documentation improvements

## What we avoid

- Presenting this as an official restaurant website
- Removing or weakening disclaimer language
- Adding misleading booking or payment flows
- Committing credentials or API keys

## How to contribute

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Commit with a clear message
5. Open a Pull Request describing what changed and why

## Code style

- Match existing naming and structure
- Keep animations respectful of `prefers-reduced-motion`
- Use SVG icons, not emoji, for UI elements
- Minimise scope — focused diffs are easier to review
